About this specification

This HTML5 specification is like no other—It has been processed with you, the humble web developer in mind.

We've focused on readability, and ease of access. That is, you should find a few spare hours and read it from zero to hero.

The design is derived from work by Ben Schwarz and Anthony Kolber, of AB+C, enabled by the legendary Mike™ Smith, W3C.
Read about how it all started.

If you spot something that you feel isn't quite right, please raise an issue, otherwise you'll find the source on github.

This version is maintained by Ben Schwarz and Ian Hickson.
