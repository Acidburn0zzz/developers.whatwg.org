About this specification

This HTML5 specification is like no other—It has been processed with you, the humble web developer in mind.

We've focused on readability and ease of access. Unlike the full HTML specification, this "web developer edition" removes information that only browser vendors need know.

This edition is written by Ian Hickson, everything style and presentational, by Ben Schwarz and is a derivative of work that was conducted by Ben, Anthony Kolber (Together, forming ab+c) and Mike™ Smith, W3C.
